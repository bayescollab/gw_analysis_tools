Detailed Description

Simpson's rule for logarithmically uniformly-spaced integrals.

Quadrature with the classic extended 3-point rule (see, e.g., Numerical Recipes, extended Simpsons rule). For even lengths, the trapezoidal rule is used at the last interval.
