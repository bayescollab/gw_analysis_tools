Internal function to launch the CUDA kernel for a range of autocorrelations.

Correlation function used:

rho(lag) = 1 / (length - lag) \sum (arr[i+lag]-average) ( arr[i]- average)

target_corr = rho(rho_index)/rho(0) = rho(rho_index)/var

Parameters
arrInput array of data
lengthLength of data array
averageAverage of input data
[out]rho_indexIndex of the lag that results ina correlation ratio target_corr
target_corrTarget correlation ratio rho(lag)/rho(0) = target_corr
varVariance rho(0)
start_idStarting index to use for the data array arr
