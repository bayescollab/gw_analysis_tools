Detailed Description

File for the addition of tidal effects to the precessing waveform model IMRPhenomP.

Extends the IMRPhenomP template to include tidal effects, through inheritance from IMRPhenomD_NRT, following the NRTidal model of arXiv:1905.06011v2. Specifically, equations 17, 18, 19, 20, 21, and 24 were used.
